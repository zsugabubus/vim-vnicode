*vnicode.txt*	Show Unicode character under the cursor.

					Type |gO| to see the table of contents.

==============================================================================
INTRO							*vnicode-intro*

This plugin shows Unicode character under the cursor (|ga|) and
provides ability to search Unicode characters by their name and looking up
miscalleous information using |i_CTRL-V_/|.

The above mentioned keystrokes are overmapped only if they are unmapped.
Alternatively, you can use |vnicode#show()| or open buffer
|vnicode://NamesList.txt| for searching respectively.

NOTE: Plugin downloads required files (currently UnicodeData.txt and NamesList.txt)
on demand into "/path/to/plugin/data/".
